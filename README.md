# windows 客户端

> 简单理解的话就是因为懒得去命令行敲命令，所以才实现了这个 GUI 工具库

## 功能 1

alt + 空格 在屏幕中间出现搜索框，进行跨字符的模糊搜索，按下回车键使用 vscode 打开项项目

![demo](./z_demo/demo.png)

## 功能 2

alt + v 唤起一个窗口，点击后直接输入进输入框 (相当于快速 `ctrl+c` 和 `ctrl+v`)

> 在设置中保存的笔记列表在这里使用

![demo](./z_demo/quickInput.png)

## 功能 3456

- 杀死端口号
- 用窗口打开网页
- 执行一些常用命令
- 删除指定目录下的所有文件 (例如 `node_modules`, 嗯, 虽然好像没快多少)

![demo](./z_demo/small-tool.png)

## 使用前需要设置一些基本数据

入口在托盘图标 鼠标右击 -> 设置

![init](./z_demo/init.png)
